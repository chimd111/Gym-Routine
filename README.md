# 5 Day Gym Routine

A sleek, single-page gym routine tracker with a dark, neon-accented design. Each weekday is an expandable card listing exercises with direct links to YouTube tutorials.

## Features

- **Single-page layout** — all five days live on one page, no extra navigation needed
- **Expandable day cards** — click any day to reveal its exercises, click again to collapse
- **Scroll animations** — cards fade in smoothly as you scroll down the page
- **Direct exercise links** — every exercise links out to a YouTube search for tutorials
- **Set tags** — each exercise shows a quick reference for recommended sets



### How to Use

1. Open `routine.html` in your browser.
2. Click any day card to expand it and see that day's exercises.
3. Click an exercise to open a YouTube tutorial search in a new tab.

## File Details

- **`routine.html`** — the main (and only) page, containing the hero section and all five day cards
- **`styles.css`** — all styling, including the dark theme, glass-effect cards, fade-in animations, and exercise list styling
- **`script.js`** — handles the expand/collapse behavior for day cards and the scroll-triggered fade-in animation
