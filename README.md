# 5 Day Gym Routine

This project is a simple HTML and CSS-based website that provides a 5-day gym routine. Each day focuses on a specific muscle group, with links to relevant exercise tutorials.

## Features

- **Daily Routine Pages**: Each day has its own HTML page with a list of exercises and links to YouTube search results for tutorials.
- **Responsive Design**: Styled using CSS for a visually appealing and user-friendly experience.
- **Grid Layout**: The main page uses a grid layout to display links to daily routines.

## How to Use

1. Open `index.html` in your browser to view the main page.
2. Click on any day to navigate to its specific routine page.
3. Each routine page contains a list of exercises with links to YouTube tutorials.

## File Details

- **`index.html`**: The main page with a grid layout linking to daily routines.
- **`daystyles.css`**: Styles for the individual day pages.
- **`style.css`**: Styles for the main page, including grid layout and day-specific styles.
- **Daily HTML Files**: Each file (`monday.html`, `tuesday.html`, etc.) contains the routine for a specific day.
